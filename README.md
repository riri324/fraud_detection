# Synthetic Data Generation using CTGAN

This repository contains the code, experiments, and documentation for my final year project on synthetic data generation using CTGAN.

## Repository Structure

* **experiments/**
  Contains experimental notebooks used for model training and evaluation.

* **Development_Logs/**
  Contains screenshots documenting the development history from external platforms such as Kaggle and Google Colab.

* **Research_Findings-4.pdf**
  Supporting research notes and intermediate findings. This document summarises background literature and potential     modelling approaches explored during the early stages of the project, and was used to present and discuss the project direction with the supervisor.

* **Brief_Project_proposal.pdf**
  Initial project proposal submitted at the beginning of the dissertation.

## External Development Logs

During development, many experiments were conducted on external platforms such as **Kaggle** and **Google Colab**, primarily to utilise GPU resources and quickly iterate on model configurations.

Screenshots of the version histories from these platforms are included in the `Development_Logs` folder to demonstrate the iterative development process and experimentation history.

These logs show the progression of experiments leading to the final CTGAN implementation included in this repository.
