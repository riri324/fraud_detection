# Development Logs

This folder contains screenshots documenting the development history of the project.

## Kaggle Development

A large portion of experimentation for this project was conducted using Kaggle notebooks. Kaggle was primarily used as an environment for running and managing machine learning experiments.

The screenshots included in this folder show the version history of several Kaggle notebooks, demonstrating the iterative development process across multiple experiments.

Experiments conducted on Kaggle include:

* Training and testing **CTGAN** models for synthetic data generation
* Training and testing **Findiff** models for synthetic data generation
* Implementing **XGBoost baseline models** for comparison with synthetic data experiments
* Testing different preprocessing strategies and dataset configurations

These logs illustrate the progression of the project through multiple iterations, where different model architectures, parameters, and evaluation strategies were tested and refined.

## Google Colab Development

Google Colab was used during later stages of development, particularly when additional GPU resources were required for model training and experimentation.

The screenshots included here show the revision history of the notebooks used on Colab. These revisions document code modifications, debugging steps, and improvements made to the training pipeline.

Activities performed on Colab include:

* GPU-based training for GAN and diffusion experiments
* debugging model training issues
* refining model parameters and evaluation workflows

## Purpose of These Logs

Since a significant portion of experimentation was conducted on external platforms such as Kaggle and Google Colab, these screenshots are included to document the development process.

They demonstrate the iterative experimentation and model development that led to the final implementation included in this repository.
